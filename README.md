🎮 Rock · Paper · Scissors — Premium UI Game

A beautifully designed Rock, Paper, Scissors web app built using Streamlit, featuring a modern UI, smooth animations, and real-time gameplay stats.

🚀 Features

✨ Clean & aesthetic dark UI
🎯 Interactive gameplay (vs CPU)
📊 Live scoreboard (Wins / Losses / Ties)
🔥 Streak tracking system
📈 Win-rate progress bar
🕹️ Animated battle arena
📜 Recent match history (last 12 rounds)
🔄 Reset game functionality
⚡ Smooth animations & hover effects

🛠️ Tech Stack
Python
Streamlit
Custom CSS (UI/UX design)
Random module (game logic)
📂 Project Structure
📦 Rock-Paper-Scissors
 ┣ 📜 app.py              # Main Streamlit application
 ┣ 📜 config.toml        # UI theme configuration
 ┣ 📜 requirements.txt   # Dependencies
 ┗ 📜 README.md          # Project documentation
⚙️ Installation & Setup
1. Clone the repository
git clone https://github.com/your-username/rock-paper-scissors.git
cd rock-paper-scissors
2. Install dependencies
pip install -r requirements.txt
3. Run the app
streamlit run app.py
🎮 How to Play
Click on Rock 🪨, Paper 📄, or Scissors ✂️
The CPU will randomly choose its move
The result is displayed instantly
Track your:
Wins 🟢
Losses 🔴
Ties 🟡
Try to build a winning streak 🔥
